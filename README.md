# web-wechat-chrome-extension
从这个扩展fork之后，修改而得。因为老的扩展，在新版本的网页版微信不能用了。

安装方法：
1. 安装npm, yarn等组件
2. 执行：npm install
3. 执行：build.sh

当前目录下会生成build文件夹。然后用chrome加载这个文件夹即可。

功能：
1. 支持两个账号(wx and wx2)
2. 未读消息数精准通知
3. 消息防撤回
